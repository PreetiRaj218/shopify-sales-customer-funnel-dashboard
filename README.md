# shopify-sales-customer-funnel-dashboard

🛍️ Shopify Sales & Customer Funnel Dashboard | Power BI
📌 Project Overview

This project presents an interactive Shopify Sales & Customer Funnel Dashboard built in Power BI. The dashboard provides insights into sales performance, customer purchasing behavior, regional sales distribution, payment methods, and product performance.

The objective of this project is to help businesses monitor key metrics and make data-driven decisions through interactive visualizations.

## 📊 Dashboard Features
💰 Net Sales Analysis
📦 Total Quantity Sold
👥 Customer Analysis
🔄 Repeat Customer Rate
💳 Payment Gateway Analysis
🌍 Regional Sales Overview
📈 Sales Trend Over Time
🛒 Product-wise Sales Performance
🎯 Interactive Filters & Slicers
